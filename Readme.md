Wechat LuckyMoney
=====
[下载](https://raw.githubusercontent.com/MidoriYakumo/LuckyMoney/master/out/app-debug.apk)

安卓4.4以上(4.3未测试)，支持以下情形:

1. 处于本群聊天视图, 并位于最底部

2. 处于其他群聊天视图, 开启通知

3. 处于微信以外, 开启微信通知, 屏幕点亮

4. 无密码锁屏, 开启微信通知

还有很多意外情况，考虑到i18n的问题，懒得处理．
